# Implementation Plan: Focused Effect Vocabulary + Embrace Repetition

**Branch**: `036-focused-effects-repetition` | **Date**: 2026-04-09 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/036-focused-effects-repetition/spec.md`
**Constitution**: v1.0.0

## Summary

Phase 1 of Sequence Quality Refinement. Two changes to make generated sequences look
more like hand-sequenced community files:

1. **Focused vocabulary**: Algorithmically derive a weighted working set of 4-8 effects
   from each theme's layer structure. The bottom (base) layer's effect gets the highest
   weight, upper layers get decreasing weights. This replaces the current uniform selection
   from a broad pool.

2. **Embrace repetition**: Remove intra-section reuse penalties in the rotation engine
   so the same effect+palette sustains across an entire section on each model. Cross-section
   penalties are relaxed (not removed) for adjacent same-theme sections. Beat tier (Tier 4)
   is excluded — its chase pattern is unchanged.
